# LAMP Stack Application Hosting on AWS

## Overview

This project demonstrates deployment of a LAMP Stack application on AWS Cloud using:

* Linux (Ubuntu EC2)
* Apache Web Server
* MySQL Database (Amazon RDS)
* PHP

The application is hosted on an EC2 instance and connected with an Amazon RDS MySQL database.

---

## AWS Services Used

* Amazon EC2
* Amazon RDS
* Security Groups

---

## Technologies Used

* PHP
* MySQL
* Apache2
* Ubuntu Linux
* AWS Cloud
* Git & GitHub

---

## Features

* Hosted PHP web application on AWS
* Connected EC2 with RDS MySQL database
* Dynamic data fetched from database
* Apache web server configuration
* Secure cloud deployment

---

## Architecture

```text
User → EC2 (Apache + PHP) → RDS MySQL Database
```
